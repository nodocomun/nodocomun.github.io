Websites para ti y tus proyectos.

Hospedado directamente desde tu repositorio de GitHub. Sólo edita, realiza push y tus cambios se ven en línea.

Ayuda de Páginas

¿Qué son las páginas de GitHub?

¿Listo para empezar? Hazte tu propia página o genérala para tu proyecto.

Puedes tener una web por cuenta de GitHub u organización,
e ilimitadas páginas de proyectos. ¿Lista/o? Vamos a empezar.

  • Usuario u Organización
  • Web de proyecto

  • Crear un nuevo repositorio

    Ve a GitHub y crea un nuevo repositorio llamado username.github.io, donde username es tu nombre de usuario (o nombre de la organización) en GitHub.

    Si la primera parte del repositorio no es exactamente como tu nombre de usuario, no funcionará, así que asegúrate que esté bien escrito.

  • ¿Qué cliente de git estás usando?

    • Terminal
    • GitHub para Windows
    • GitHub para Mac
    • No lo sé

  • Descargar GitHub para Windows

    GitHub para Windows es una forma genial de usar git y GitHub en Windows.

    Descargar GitHub para Windows

  • Descargar GitHub para Mac

    GitHub para Mac es una forma genial de usar git y GitHub en Mac.

    Descargar GitHub para Mac

  • Clona el repositorio

    Ve a la carpeta donde quieres guardar el proyecto y clona el nuevo repositorio:

    ~$git clone https://github.com/username/username.github.io

  • Clona el repositorio

    Haz clic en el botón "Set up in Desktop". Cuando la aplicación de escritorio de GitHub abra, guarda el proyecto.

    Si la aplicación no se abre, iniciala y clona el repositorio desde la aplicación.

  • Clona el repositorio

    Después de finalizar la instalación, vuelve a GitHub.com y refresca la página. Haz clic en el botón "Set up in Desktop". Cuando la aplicación de escritorio de GitHub abra, guarda el proyecto.

    Si la aplicación no se abre, iniciala y clona el repositorio desde la aplicación.

  • Hola mundo

    Ingresa a la carpeta del proyecto y crea un archivo index.html:

    ~$cd username.github.io

    ~$echo "Hola mundo" > index.html

  • Crear un archivo index

    Escoge tu editor de texto favorito y añade un archivo index.html a tu proyecto:

    index.html
    <!DOCTYPE html>
<html>
<body>
<h1>Hello World</h1>
<p>I'm hosted with GitHub Pages.</p>
</body>
</html>

  • Push it (Publicar)

    Añade, haz commit, y push en tus cambios:

    ~$git add --all

    ~$git commit -m "Initial commit"

    ~$git push -u origin master

  • Commit & sync

    Entra en el repositorio, haz commit a tus cambios, y presiona el botón "Sync".

  • Commit & sync

    Entra en el repositorio, haz commit a tus cambios, y presiona el botón "Sync".

  • …¡y ya está!

    Inicia un navegador y ve a http://username.github.io.

  • ¿Utilizar un tema, o empezar desde cero?

    Tienes la opción de empezar con uno de los temas prestablecidos,
    o crear un sitio desde cero.

    • Escoge un tema
    • Empezar desde cero

  • Opciones de los repositorios

    Ve a GitHub.com y crea un nuevo repositorio, o directamente ve a uno que ya exista.
    Haz clic en la pestaña Settings.

  • Selector de tema

    Baja a la sección de GitHub Pages. Presiona Escoger un tema.

  • Escoge un tema

    Elige uno de los temas del carrusel arriba.
    Cuando estés listo, haz clic en Seleccionar tema a la derecha.

  • Editar contenido

    Usa el editor para añadir contenido a tu sitio

  • Commit

    Ingresa un comentario de commit y hac clic en Commit changes debajo del editor.

  • Crea un archivo de index

    Ve a GitHub.com y crea un nuevo repositorio, o selecciona uno que ya exista.
    Haz clic en el botón Crear un nuevo archivo.

  • Hola mundo

    Nombra al archivo index.html y escribe algo de contenido HTML en el editor.

  • Commit el archivo

    Baja al final de la página, escribe un mensaje de commit, y realiza un commit del nuevo archivo.

  • Opciones del repositorio

    Haz clic en la pestaña de Settings y baja a la sección de GitHub Pages.
    Después selecciona la raíz de master branch y haz clic en el botón Save.

  • …¡y ya está!

    Inicia un navegador y ve al repositorio de http://username.github.io/.
